Class Persona

Description
Description | Methods (details)

Script para manejar los datos de una persona.

Este script define una clase Persona con métodos para establecer y obtener los datos básicos de una persona, como su nombre, edad y género. También se incluye un método para mostrar los datos de la persona en formato HTML.
